Exposure Commands:

Back to overview
set_backlight_comp

Turn the backlight compensation on (1) or off (0)
set_whitebal_mode

Set the whitebalance mode to 0: Auto, 1: Indoor, 2: Outdoor, 3: OnePush
set_whitebal_one_push

Push trigger for whitebalance in OnePush mode
set_auto_exp_mode

Set the AE mode to 0: Full Auto, 3: Manual, 10: Shutter priority, 11: Iris priority, 13: Bright Mode
set_shutter_value

Set the shutter value between 0: 1/60 and 27: 1/10000
set_shutter_up

Increase the shutter speed, available only with shutter_priority or AE_Manual
set_shutter_down

Decrease the shutter speed, available only with shutter_priority or AE_Manual
set_shutter_reset

Reset the shutter speed, available only with shutter_priority or AE_Manual
set_iris_value

set the iris opening to a value between 0: closed and 17: F1.8
set_iris_up

Open up the iris, available only with iris_priority or AE_Manual
set_iris_down

Close the iris, available only with iris_priority or AE_Manual
set_iris_reset

Reset the iris, available only with iris_priority or AE_Manual
set_gain_value

set the gain value between 1: 0dB and 7: +18dB
set_gain_up

Increase the gain, available only with AE_Manual
set_gain_down

Reset the gain, available only with AE_Manual
set_bright_up

brighten the image, available only with bright_mode
set_bright_down

Darken the image, available only with bright_mode
set_bright_reset

Reset the image brightness, available only with bright_mode
