Auto tracking and motion detection Commands:

Back to overview
set_at_mode

Set the target tracking mode to On (1) or Off (0)
set_at_mode_onoff

Toggle target tracking mode
set_md_mode

Set the motion detection mode to On (1) or Off (0)
set_md_mode_onoff

Toggle motion detection mode
set_atmd_startstop

Start or stop target tracking or motion detection
set_at_ae

Set the auto exposure for target tracking mode to On (1) or Off (0)
set_at_ae_onoff

Toggle auto exposure for target tracking mode
set_at_autozoom

Set the auto zoom for target tracking mode to On (1) or Off (0)
set_at_autozoom_onoff

Toggle Auto zoom for target tracking mode
set_atmd_framedisplay

Set the frame display for target tracking or motion detection mode to On (1) or Off (0)
set_atmd_framedisplay_onoff

Toggle frame display for target tracking or motion detection mode
set_at_frameoffset_onoff

Toggle frame offset control for target tracking mode
set_md_frame

Set detection area/size for motion detection
set_md_detect

Switch detecting frame for motion detection: 1, 2 or 1+2
set_md_measure_mode1

Set the motion detection measure mode 1 to On (1) or Off (0)
set_md_measure_mode1_onoff

Toggle motion detection measure mode 1
set_md_measure_mode2

Set the motion detection measure mode 2 to On (1) or Off (0)
set_md_measure_mode2_onoff

Toggle motion detection measure mode 2
set_at_chase_next

Switch target tracking chase mode 1/2/3
set_at_chase
chase_mode:
Select target tracking chase mode 0, 1 or 2
set_at_entry
entry:
Select target study mode for target tracking. Available modes: 0 to 3
set_md_adjust_ylevel
level:
Adjust level of Y-Level motion detection between 0 and 15
set_md_adjust_huelevel
level:
Adjust level of Hue-Level motion detection between 0 and 15
set_md_adjust_size
level:
Adjust level of size motion detection between 0 and 15
set_md_adjust_disptime
level:
Adjust level of disptime motion detection between 0 and 15
set_md_adjust_refmode
mode:
Set refreshmode for motion detection from 0 to 2
set_md_adjust_reftime
time:
Set refreshtime from 0 to 15
