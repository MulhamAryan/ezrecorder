General Commands:

set_power

Set the power state to on (1) or off (0)
set_irreceive_on

Enable IR remote controller
set_irreceive_off

Disable IR remote controller
set_irreceive_onoff

Toggle IR remote controller
set_datascreen_on

Show data display
set_datascreen_off

Hide data display
set_datascreen_onoff

Toggle data display
set_keylock

Set the keylock to on (1) or off (0)
memory_set
channel:
Save the current position to a channel between 0 and 5
memory_recall
channel:
Recall the current position from a channel between 0 and 5
memory_reset
channel:
Reset a channel between 0 and 5
set_wide_con_lens
conversion:
AT compensation when a wide conversion lens is installed, 0: No conversion to 7: X0.6 conversion)
